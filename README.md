# ItamiProject
Работа в этом репозитории окончена, скорее всего, он больше не будет обновляться.

В этом небольшом учебном проекте для университета мы разработали небольшую игру на Windows Forms на языке C#. Здесь мы постарались **реализовать архитектурный паттерн MVC.** А ещё, мы научились работать в команде и работать с git. Игра готова. Вы можете скачать и поиграть в неё в разделе [Releases](https://github.com/romankuvshanov/ItamiProject/releases).
